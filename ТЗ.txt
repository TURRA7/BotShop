Задание: Сделать телеграм бота - магазин, в котором будут следующие разделы:

У пользователя:
1. Каталог - в нём будут содержаться товары, у которых будет кнопка купить.
* Товар имеет: название, описапние, цена, количество на складе, картинка.
2. Корзина - В ней будут находится цифровые товары(если пользователь их выбрал),
например гифты, итоговая стоимость, кнопка удаления конкретного товара,
кнопка оплаты и кнопка добавление реф кода.
3. Выбор оплаты - тут должны быть варианты оплаты и подключенные к ним сервисы,
после оплаты пользователю должен быть выдан его товар, например сгенерированный
код из букв и цифр(это и будет гифт).
4. Баланс - он должен показывать текущий баланс, иметь кнопку пополнить,
а так же перевести деньги другому пользователю.
5. Реферальная система - пользователь получает уникальный код, он может
дать его своему другу и тот введя его при оплате в корзине, получит -5% к цене.
6. Контакты - связь с оператором, связь с администрацией.

У администратора/оператора + к основному меню, кнопка с админ-меню:
1. Добавление товара - запускает FSM машину с процессом добавления товара.
2. У каждого товара в каталоге должна быть кнопка "УДАЛИТЬ".
3. Условная кнопка "Сгенирировать код" - для собственно-ручной выдачи кода.
4. Пополнить аккаунт вручную.
5. Списать деньги вручную.
6. Оформления возврата.


Важные детали:
  * Все операции должны передаваться в 'менеджер операций', дабы при 
    падении контейнера с приложением, задачи не исчезали и пользователи
    не теряли деньги.
  * Бот, Базы данных и другие сторонние сервисы, должны быть упакованны
  в docker контейнеры и должны подниматься с помощью docker-compose.
  * Должна быть внятная документация по каждому модулю.
  * В качестве фреймворка для бота, должен быть использован Aiogram3.
